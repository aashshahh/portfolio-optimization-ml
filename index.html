// Architecture

Three-stage
ML Pipeline

Each stage feeds the next — regime detection informs covariance estimation, ML predictions define the expected return vector, and MVO constructs the final portfolio.

01 / 03
🧠
Regime Detection

GaussianHMM with 3 latent states learns Bear/Sideways/Bull market regimes from SPY's return, volatility, momentum, and skewness — without any labels.

hmmlearn GaussianHMM Viterbi 3-state
02 / 03
📡
Return Forecasting

LightGBM trained on 14 features per stock: momentum factors (1d–252d), RSI, vol ratio, beta, and regime dummies. Walk-forward validation prevents look-ahead bias.

LightGBM walk-forward IC metric SHAP
03 / 03
⚖️
MVO Construction

Markowitz mean-variance optimization with ML-predicted μ and regime-conditioned Σ (Ledoit-Wolf shrinkage). Monthly rebalancing, max 15% per stock.

scipy SLSQP Ledoit-Wolf monthly rebal no shorting
